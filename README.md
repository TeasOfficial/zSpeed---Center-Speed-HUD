# zSpeed---Center-Speed-HUD
Center Speed HUD for CS:S Bunny Hop 

Original author: Shahrazad

Modified & Rewrite by: Picrisol45

You can use dynamic colors, adjust HUD color values, and change the refresh rate.

NOTE: Display Speed Difference HUD may BREAK other HUDs(JHUD, StrafeTrainer), so don’t open it unless necessary!!!
